# بنية الصفحات (Pages Structure)

## الصفحات الرئيسية

### 1. الصفحة الرئيسية (Home/Chat)
- **المسار**: `/`
- **الملف**: `app/(chat)/page.tsx`
- **الوصف**: صفحة المحادثة الجديدة

### 2. صفحة المحادثة المحددة
- **المسار**: `/chat/[id]`
- **الملف**: `app/(chat)/chat/[id]/page.tsx`
- **الوصف**: عرض محادثة محددة حسب المعرف

### 3. تسجيل الدخول
- **المسار**: `/login`
- **الملف**: `app/(auth)/login/page.tsx`
- **الوصف**: صفحة تسجيل الدخول للمستخدمين

### 4. التسجيل
- **المسار**: `/register`
- **الملف**: `app/(auth)/register/page.tsx`
- **الوصف**: صفحة إنشاء حساب جديد

### 5. لوحة الإدارة
- **المسار**: `/admin`
- **الملف**: `app/admin/page.tsx`
- **الوصف**: لوحة تحكم الإدارة

### 6. إدارة الوكلاء
- **المسار**: `/admin/agents`
- **الملف**: `app/admin/agents/page.tsx`
- **الوصف**: إدارة وكلاء الذكاء الاصطناعي

### 7. السوق
- **المسار**: `/marketplace`
- **الملف**: `app/marketplace/page.tsx`
- **الوصف**: سوق الوكلاء والخدمات

## مسارات API

### المصادقة
- `/api/auth/[...nextauth]` - NextAuth.js handlers
- `/api/auth/guest` - إنشاء مستخدم ضيف

### المحادثة
- `/api/chat` - إنشاء محادثة جديدة
- `/api/chat/[id]/stream` - بث المحادثة
- `/api/history` - تاريخ المحادثات
- `/api/suggestions` - الاقتراحات
- `/api/vote` - التصويت على الرسائل

### الملفات والمستندات
- `/api/files/upload` - رفع الملفات
- `/api/document` - إدارة المستندات

### الوكلاء
- `/api/agents` - قائمة الوكلاء
- `/api/marketplace/agents` - وكلاء السوق
- `/api/marketplace/agents/[id]` - وكيل محدد

## البنية الحالية

```
app/
├── (auth)/              # مجموعة مسارات المصادقة
│   ├── login/           # صفحة تسجيل الدخول
│   ├── register/       # صفحة التسجيل
│   └── api/            # API المصادقة
├── (chat)/             # مجموعة مسارات المحادثة
│   ├── page.tsx        # الصفحة الرئيسية
│   ├── chat/[id]/      # محادثة محددة
│   └── api/            # API المحادثة
├── admin/              # لوحة الإدارة
│   ├── page.tsx        # الصفحة الرئيسية
│   ├── agents/         # إدارة الوكلاء
│   └── api/            # API الإدارة
├── marketplace/       # السوق
│   └── page.tsx        # صفحة السوق
└── layout.tsx          # التخطيط الرئيسي
```

## ملاحظات

- المجلدات بين قوسين `(auth)` و `(chat)` هي Route Groups في Next.js ولا تظهر في URL
- جميع الصفحات تستخدم App Router من Next.js
- التخطيطات (Layouts) مشتركة لكل مجموعة من المسارات
